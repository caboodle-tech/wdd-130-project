Challenges

This directory and it's child directories are for the various challenges you can do throughout the semester. The instructions for these challenges will be given to you in Canvas. You will mostly interact with the challenge files in your code editor but we have pre-structured them on this website so you can easily test them. This website will also be the means of receiving credit for completing a challenge. The instructor or a teaching assistant will visit your website weekly and check that you have completed a challenge. Make sure your push your changes to GitHub often including deploying the latest version of your GitHub Pages website.

Please use the drop down navigation menus to view the challenge files online.
